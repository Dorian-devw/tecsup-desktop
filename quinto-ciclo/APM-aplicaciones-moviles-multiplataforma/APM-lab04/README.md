# APM Lab 04

Este laboratorio está resuelto usando solo Dart puro (sin Flutter) y presenta una implementación sencilla para cada ejercicio.

## Cómo ejecutar

Desde la carpeta `APM-lab04`, ejecutar cada ejercicio con:

```bash
dart ejercicio1/ejercicio1.dart
```

Reemplaza `ejercicio1` por el número del ejercicio que quieras ejecutar.

## Ejercicios incluidos

- `ejercicio1/ejercicio1.dart` - Biblioteca multimedia con clase base y subtipos.
- `ejercicio2/ejercicio2.dart` - Reservas de experiencias con reporte uniforme.
- `ejercicio3/ejercicio3.dart` - Procesamiento de pagos con contrato común.
- `ejercicio4/ejercicio4.dart` - Capacidades IoT con mixins reutilizables.
- `ejercicio5/ejercicio5.dart` - Gestor de órdenes con contenedor de acciones.
- `ejercicio6/ejercicio6.dart` - Extensión de descripciones usando herencia.
- `ejercicio7/ejercicio7.dart` - Resolución de choque de mixins.
- `ejercicio8/ejercicio8.dart` - Mini-dominio integrado con gestor y reporte uniforme.

## Notas

- Cada ejercicio está en su propia carpeta y archivo.
- El código es simple y enfocado en los requisitos de la clase de hoy.
